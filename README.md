# VisionOCR 

The `VisionOCR` is an **OCR** (Optical Character Recognition) tool that detects text in an image, extracts it, and stores it as a separate text file.

## Tools

- Python 
- OpenCV 
- PyTesseract 
- Jupyter Notebook
